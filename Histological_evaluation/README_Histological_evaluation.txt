1. Folder "statistical analysis" contains the data for Fig 3A visualization (Violin_plot.py), summary of the  statistical analysis (Analysis.xlsx; statistics was done with RealStatistics toolbox https://real-statistics.com/) and cluster folders containing data used for the statistical analysis. 

Folders Cluster_1_data etc. contain the results from the multiplication of the binary cluster mask file with the binary histological data. 

Specifically, the binary histological data contains either 1 or 0 for each of the analyzed voxel representing either the presence or absence of the viral expression. Data were obtained from a direct request from the authors of the following paper 

Rocchi, F. et al. Increased fMRI connectivity upon chemogenetic inhibition of the mouse prefrontal cortex. Nature Communications 13, 1056 (2022). https://doi.org/10.1038/s41467-022-28591-3

2. Folder "visualization" contains all data needed to reproduce Fig 3A and Fig S8 using fsleyes
